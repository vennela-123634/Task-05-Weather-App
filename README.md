Task-05: Weather App
This project is a functional web page designed to fetch and display real-time weather information.
Features
Weather Data Fetching: Retrieves weather data from a weather API based on the user's current location or a user-inputted location.
Dynamic Display: Clearly displays current weather conditions, temperature, and other relevant information.

How to Run Locally
